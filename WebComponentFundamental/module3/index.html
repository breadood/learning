Core Functionality

  1. Define HTML elements
  2. Extend existing elements

* Naming has to have "-"

Register Custom Element

  1. Create prototype: var SlickTabs = Object.create(HTMLElement.prototype), if extending use HTMLButtonElement.prototype for example
  2. Register: document.registerElement('slick-tabs'), registering custom elements
  3. Use it with: document.body.appendChild(new SlickTabs()), we can also you a tag. <slick-tabs>
  4. Use .createdCallback = function() {} to add innerHtml

Instatiate Custom Element

  • Markup in HTML
  • <new> operator, and append
  • document.createElement(), and append
  • document.body.innerHTML

Example

                var MyElementProto = Object.create(HTMLElement.prototype);

                MyElement.createCallback = function() {
                  this.innerHTML = "<My Element>"
                }

Extend Custom Element

  1. Declare in markup with "is" attribute
  2. Javascript var button = document.createElement('button', 'super-button')
  3. new Operator

Example

              var MyButtonProto = Object.create(HTMLButtonElement.prototype);

              MyButton.createdCallback = function() {
                this.innerHTML = "My Special Button";
                this.value = "Default  Value";
              }

              var MyButton = document.registerElement('my-button', {
                prototype: MyButtonProto,
                extend: 'button'
              });

              // Use <button is="my-button" />;
              // also we can

              var button = document.createElement('button', 'my-button');
              document.body.appendChild(button);

Lifecycle/Callback

  1. createdCallback: created
  2. attachedCallback: inserted in DOM
  3. detachedCallback: removed in DOM
  4. attributeChangedCallback: attributed are edited

Example

              Proto.createdCallback = function () {};
              Proto.attachedCallback = function () {};
              Proto.detachedCallback = function () {};
              Proto.attributeChangedCallback = function (attrName, oldValue, newValue) {};

Naming

  1. use the first part of the customed element as namespace
  2. Company Name: twitter-tweet-button
  3. Brand Name: kendoui-grid
  4. Application Name: wechat-button
  5. Github Username: breadood-button
