Existing Template Technology

HTML in Script Tag

<script type="text/custom-name-here">

Nothing inside runs or renders

Not DOM Element

Hidden DOM Elements

<div style="display: none;" id="my-template" >

Easy to clone, DOM element

Everything runs, may throw 404

HTML5 Template

Inert

Does not run, anything will not run

Hidden From Selectors

Cannot be selected/traversed by ordinary method

Flexible Placement

Can be placed anywhere

Basic Definition/Activation

  1. Declare a <template> tag
  2. Obtain this tag by calling var template = document.querySelector()
  3. Clone the template by document.importNode(template.content, true)
  4. Insert it by document.body.append() the clone

Injecting Dynamic Data

  1. Get the reference to the template
  2. Clone the template by using document.importNode
  3. Change the target element within template dynamically
  4. Append cloned template

Nested Templates

Nested templates need to be cloned/activated each independently.
