Shadow Root HTML Method

These Normal HTML method will NOT work for shadown DOM

  1. document.querySelector
  2. document.querySelectorAll
  3. document.getElementById
  4. document.getElementsByClassName
  5. document.getElementsByTagName

We should use

  1. root.querySelector --> root is the shadow root, not the shadow host
