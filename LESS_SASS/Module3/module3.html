SASS and SCSS

Functionality

  • Variable
  • Operation
  • Color Function
  • Math Function
  • String Interpolation
  • Hierachy Structure
  • Similar to LESS

Directives

@import

import ".scss" file by @import "file". This is better because it compiles css instead of making round trips to servers.

@extend

Use extend to inherit defined rules. We can also use multiple inheritage.

@mixin

Mixins are like functions, use @include mixin-name(params). Declare mixins with parameters and default values.

@function

Define function use @function name(param) {@return value;}

Controls

  • @if/@else/@else if
  • @for var from # through ##
  • @each var in xxx {}
  • @while condition {}
