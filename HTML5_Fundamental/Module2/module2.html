Selection API

getElementsByClassName

document.getElementsByClassName('className') will return [elements...]

querySelector

document.querySelector('CSSSelector') return the first matched result

querySelectorAll

document.querySelectorAll('CSSSelector') return all matches in []

getElementsByClassName

This is a similar method to querySelectorAll, but this will return "live" result, and querySelectorAll returns "static" result. Have to recall querySelectorAll to update result.
